### File Structure
--------------------------
sbom directory should have the following file structure:
```
sboms/
├── javascript/
│   ├── diff/
│   └── raw/
└── python/
    ├── diff/
    └── raw/
```
The general structure is 
```
sboms/
└── <language>/
    ├── diff/
    └── raw/
```

Repo Directory should have the following file structure:

```
repos/
|--<languages>/
|    |--<repos>/
```
